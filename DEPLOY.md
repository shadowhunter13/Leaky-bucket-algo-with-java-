## OR: Deploy on Railway.app (3 minutes)

1. Go to [railway.app](https://railway.app)
2. Create account ‚Üí "New Project" ‚Üí "Deploy from GitHub"
3. Select your repo
4. Railway auto-deploys
5. Get live URL automatically
6. Share: `https://your-project.up.railway.app`

## OR: Deploy on Render (3 minutes)

1. Go to [render.com](https://render.com)
2. "New" ‚Üí "Web Service" ‚Üí Connect GitHub
3. Select your repo
4. Environment: `Java`
5. Build command: `javac -d . com/leakybucket/**/*.java`
6. Start command: `java -cp . com.leakybucket.simulation.BucketSimulation`
7. Deploy!
